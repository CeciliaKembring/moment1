Bootstrap

Vad är Bootstrap?

Bootstrap är ett färdigt, kostnadsfritt ramverk för utveckling av responsiva webbplatser.

Bootstrap innefattar HTML, CSS och JavaScript och har färdig kod för exempelvis knappar, tabeller och navigering.

Länk till Bootstraps webbplats

Fördelar

  • Det går snabbt och smidigt att utveckla i Bootstrap.
  • Det är ett enkelt ramverk att lära sig.
  • Det är responisvt vilket även underlättar för alla skärmar och gör det mer tillgängligt.
  • Man får snabbt resultat.

Nackdelar

  • De förbestämda klasserna som utgör Bootstrap kan resultera i mycket kod
  • Eftersom Bootstrap är populärt resulterar det även i att många webbplatser har liknande utseende.
