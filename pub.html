Arbetssätt

Eftersom det var första gången jag använde Netlify så började jag med att skapa ett konto, som var kopplat till mitt GitHub. Jag hade vid momentets början skapat ett repo i Git som sedan inhämtades via Netlify från GitHub.

Webbplatsens fick ett autogenererat namn men jag ville att webbplatsens namn skulle passa innehållet.

För att ändra namnet gick jag in på inställningar av webbplatsen i Netlify och ändrade där namnet till "moment1ramverk".

Vid varje uppdatering av mitt repo så ändrades även webbplatsen. Innan jag slutförde uppgiften säkerställde jag att webbplatsen var komplett.

Fullstack-utveckling med ramverk.
